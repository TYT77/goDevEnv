# USAGE

### How do I download and launch it locally?

```terminal
git clone https://github.com/yukke23/goDevEnv.git
cd goDevEnv
docker-compose up -d --build
```

### Required Environment
* Git
* DockerDesktop

#### windows以外の環境の場合以下のコマンドが実行できる環境であれば代替可能
* git
* docker
